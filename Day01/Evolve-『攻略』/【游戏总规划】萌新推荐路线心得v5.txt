第一步“标准311”

推荐路线原文：

“标准宇宙
>>核炸三次凑够250+质粒 别花了
>>播种一次，点上形态发生，重组，先祖，把质粒花到剩余250
>>炸黑洞去邪恶宇宙（一定要把毁灭战士这个成就做了，地狱不派兵，造一个太空驻军，为世界对撞机供能）”

“标准311”即标准宇宙3爆1播1黑洞
基本目标：
1. 熟悉基本游戏流程
2. 通过3爆1播在保留280以上质粒的前提下点出先祖（共消耗155质粒），解锁通灵塔，然后开始黑洞
3. 在黑洞末期，点击奇异灌注进行重置之前，花完所有质粒，务必点出“基因解锁”、“基因硬化”，剩余质粒加点推荐详见第二部分。
4. 黑洞完跑路前往邪恶宇宙

常见问题：
1.我为啥要重置？我玩了一整天重置了不是白玩吗？
这个游戏的流程非常非常长，可能远比你想象的长。每次重置都会获得相应的威望资源，为你的发展提供必不可少的助力。
核爆仅仅是游戏最开始的第一阶段，到后面每个阶段对于物资的需求都会有质的飞跃，没有威望资源的支持举步维艰。
（当然你也可以头铁一周目打通到最后，这可能会花你半年以上乃至淹死）

2. 游戏里有好多东西看不懂，不知道某些资源的用处，不知道某种科技如何解锁等，应该哪里去查？
游戏右下角，有相关的wiki，活用“ctrl+F”检索功能，可以很方便地找到你需要的内容
wiki隔壁的进化攻略本里有更加详尽的解释，我在此不一一列出，如果看了仍有疑问可以来群里询问。

3.我为什么在3爆1播之后要黑洞离开标准宇宙？留在这不好吗？
首先要说明的是本游戏的【成就机制】，当你点完【基因解锁】和【基因硬化】之后，就可以通过完成成就获得相应的【精通】，
【精通】是一种非常简单粗暴地独立乘算增加你全产的属性，当你完成一定成就之后，你可以在你任意资源的加成栏中找到相应的精通加成。
当你在标准宇宙外，任何一个宇宙完成成就时，都会同时视为你在【标准宇宙】完成了该成就，换而言之如果你留在标准宇宙，做成就事倍功半。

4. 生命播种飞船好寄吧难造啊！
【主观经验】这个问题其实不止是标准311阶段，在后面很长一段时间都会困扰玩家。
造飞船要注意以下几点：
①量子制造（约465K知识，与种族特质有一定关系）出来之前尽量不碰任何纳米管相关的建筑，
即星际船坞虽然很早就研制出来了，但是量子制造之前不要去建造，更不要碰里面的生命播种飞船。
②充足的前置产能准备
作为参考，推荐配置为：地面+行星工厂20，采矿前哨≥6，采矿无人机≤2，生活区和通灵塔越多越好，至少双8乃至双10（第一次播种没有的话就不用了）
③政府的选择
公司官僚主义可以大幅度增加工厂的产能，同时不增加原材料的消耗，相当于在节省时间的同时，减少相当大一部分中子的开销。

5.政府如何选择？
【主观经验】仅做一个思路参考：
①专制
在游戏初期威望资源非常低的情况下，战斗收益占据了非常大的比重，“专制”带来不小的战斗力加成，可以以更少的士兵进行相应的战斗，
从而降低战斗损耗，增加战斗频率，提升战利品获取的效率；
②民主
“民主”则走的是增加全产量的另一种选择，靠自己的资源建筑发家，优点是知识获取更快，在低士气的前提下，民主增加的士气收益极高。
③技术官僚主义/公司官僚主义
当核爆中期“技术官僚主义”研究完成后可以考虑转为该体制，可以一直用到播种后期。播种期为了完成统一，期间建议转为联邦制。
当量子制造完成准备造船时，转为“公司官僚主义”。
④联邦制/神权
而黑洞期全产量更加重要，作为“标准311”时期的玩家，“联邦制”提供的加成更高，可以考虑联邦制。
当威望资源足够，且“牧师”一系列CRISPR技能足够高时，神权对全产量的加成将大幅度提高，并且明显超越联邦制。
此处建议点出“敬拜”乃至“助手”后，黑洞期改用神权制。

>>播种*6（尽量涵盖各种星球类型，气候，统一方式，二次进化等连续成就，无钢（鲨鱼），无趣（独角兽））

>>人类+树人+独角兽，黑洞（建议精通80-100+）

基本目标：
1. 推荐路线写得非常明白，刷成就，精通到达一定程度就从邪恶宇宙跑路。
2. 在邪恶宇宙黑洞跑路前，请确保点出“突变”（遗传记忆-阿尼姆斯-神圣的回忆-黄金分割比-基因库-突变）
3. 前往标准宇宙进行衰变挑战。
4. 衰变完前往反物质宇宙。

简单说明：
1.关于金星挑战
①何为金星挑战？
金星挑战开启方式为，在DNA进化阶段末期即将感应进化出具体种族时，可以选择相应的挑战基因，
在标准、邪恶宇宙共有“关闭初始质粒”、“关闭手动锻造”、“关闭手动贸易”、“垃圾基因”四大挑战基因，
开启0-4个挑战基因分别对应无星、白星、铜星、银星以及金星。邪恶宇宙金星铺成就即全程保持开启全部四大挑战基因。

②为什么要进行金星挑战？
在邪恶宇宙铺成就的过程中，获得的奖励与当前周目的挑战等级直接挂钩，金星挑战获得金星成就，银星挑战获得银星成就以此类推。
金星成就获得的精通数值是无星成就的五倍之多，部分成就解锁后将解锁相应的特权，特权效果同样大多数情况下金星特权效果是无星特权的五倍。
所以在邪恶宇宙铺成就的过程中，默认全部都以金星进行挑战。

2.关于脚本
邪恶宇宙阶段将不像第一阶的全新开荒，游戏内容开始了重复且冗长的循环，手操可能会变得非常乏味，一直坚持手操的玩家个人建议可以考虑开始使用脚本了。
脚本的主程序和安装方法在群文件中，开始使用脚本不等于一般单机游戏开风灵月影，而是这个游戏的一个新世界的展开。
在群内的群文件里有详细的安装图文教程，可以查看自行安装，如果仍有问题可以在群内提问。

3.关于游戏规划
①在萌新推荐路线中，“路人种族核爆9次”、“播种6次”仅仅是在绝对高效完美完成成就，在极端合理且幸运的情况下能完成保底精通的重置次数，实际是绝对做不到的。
实际游玩过程中，不用拘泥于核爆和播种的次数，只需要尽量高效地规划精通获取的效率，达到一个你认为合理的精通数值，然后从邪恶宇宙跑路即可。
②以本人游戏经历作为参考的话，比较急着体验新内容的玩家在合计85%精通时可以跑路，比较稳健的选手可能会选择达到100%+之后再跑路，十里坡玩家听说甚至有160%精通跑路的。
这是个人选择的问题，并无绝对正确的答案，当然建议别50%都不到就跑了，那后续可能要坐牢了（笑）

4.【主观经验】关于邪恶宇宙的毕业精通问题
最近很多玩家对于邪恶宇宙是否需要全爆全播的问题进行了讨论，邪恶宇宙到底是精通拉满再走还是快速跑路？我提出个人看法：
——建议快速跑路。
①邪恶不多刷一点后面不会坐牢吗？
不会。萌新推荐路线其实可以理解为一个“养成路线”，它全程绝大多数过程其实都是“成长历程”，
换句话说，一直到高引的后半段开始真正迎接挑战之前你都是“发育期”，你到邪恶还是高引无非是从一个“低级练级地”到了“高级练级地”，不存在前面刷少了后面坐牢的说法。
②为什么邪恶不建议多刷一点？
因为本游戏的【主宇宙】可以认为是高引力宇宙，将来大多数的刷资源、挑战都是在高引力宇宙展开的，所以高引力宇宙的全爆全播是必须要进行的，那么一个问题就引出来了：
所有邪恶宇宙完成过的成就到高引力宇宙还需要重复完成一遍。重复完成的成就只加本宇宙精通，相当于一个成就的本宇宙精通和全宇宙精通你分成了两次金星重置才完成。
邪恶宇宙完成的成就越多，这种重复劳动也就越多。从总时间上来看，邪恶宇宙做得越多，总消耗就越大。
③不要对高引力宇宙过于害怕
高引力宇宙其实相对邪恶宇宙并不会难太多，燃料消耗的增加其实只要有一定的反质粒，并且点出了“维度扭曲”（噬菌体增加储量上限），并不会过于增加游戏难度。
因此你完全可以把高引力宇宙理解成“高级练功房”，邪恶宇宙理解成“低级练功房”，完全没有必要在低级练功房里浪费太多时间，85%左右合计精通就可以结束邪恶宇宙。
④“强迫症”——成就收集问题
有些玩家认为成就不刷满就浑身不舒服，这一点我还是推荐根据玩家自身喜好，想怎么玩就怎么玩，只是我单纯建议一下，从功利角度来说，先快速完成萌新推荐路线，
然后在拥有极高的威望资源和特权加持后再考虑全成就收集可能会更快一点。

常见问题：
1. 霉菌瓦尔迪啥玩意？霉菌人和瓦尔迪吗？
霉菌瓦尔迪是一个种族，在进化时选择霉菌种族，并在感应进化出种族之前，选择剧情模式“遗传绝境”。

2. 我需要金核爆多少次再开始金播种？
这个问题没有正确答案，如果以“最少重置次数”、“最少现实时间消耗”以及“最高效完成成就获得精通”为标准的话，金爆和金播应当是穿插着来的，
即到一个星球，将其特有种族金爆一遍之后再选一个金播。但是从“游戏体验”上来说，先大量金核爆积攒一定量精通后再金播，会舒服非常多。
此处玩家可以自己权衡，没人催你赶进度，可以自己怎么舒服怎么来。

3.播种完给我的星球好烂啊，但是为啥SL没用？
星球刷新机制参见WIKI“随机种子机制”，比较简单的说，本阶段的玩家想要得到新的星球的办法有三种：
① 造更多的星际探测器（获得更多的星球选项但是不会刷新原有结果）
② 等待一个随机事件后刷新结果（例. “有人在你家门口扔了一袋咸鱼并点燃”）
③ 花质粒更换总督后刷新结果（空间推理-地理学家-建筑师-总督）
 
4. 为啥“无趣”挑战要用独角兽，“无钢”挑战要用鲨鱼人？
①独角兽【极乐】特质可以有效规避低士气的产量惩罚，同时【优美】特质带来的圣地可以增加士气，缓解没有娱乐建筑带来的士气低迷；
②鲨鱼人【掠食者】、【嗜血】特质极其擅长战斗，当钢铁无法通过冶炼厂产出时，战斗将成为钢的主要来源之一。
同时鲨鱼人涉及“激光鲨鱼”、“海洋开化者”、“马达加斯加树”等成就，非常适合拿来做无钢播种成就

5.牧师有啥用？能给我加血上buff吗？要点这个技能吗？
牧师的作用是加强你的宗教建筑，包括寺庙以及通灵塔。初级牧师的作用并不会太明显，同时会增加你的工作人口压力。随着CRISPR中牧师等级的提升，其效果会越来越明显。
至于加点方面并没有一个定论，我此处给予我个人的建议，至少能保证在独角兽出邪恶宇宙时点出突变的前提下，我自己的牧师点到“敬拜”。

6.CRISPR技能加点推荐
crispr技能效果以及费用在wiki中都可以查阅，安装统计脚本之后可以观看详细的技能树，此处给出一个大致的加点推荐。
①“灵巧”系列（【技工】-【注重细节】-【一丝不苟】）
能够直接、大幅度增加自动锻造速度，并且成本不高，是加成效果性价比最高最明显的第一梯队技能。
②【谈判专家】和【宇宙共同体】
二者属于位于第二梯队，·前者可以使市场交易收入受到精通增幅，后者增加本宇宙精通的加成。
③“信仰”系列（【先祖】-【信仰】- ...... ）
【信仰】的作用是解锁“牧师”，牧师的用途是增加寺庙的信仰加成，在神权体系下增加通灵塔的加成。
牧师在解锁后将会占据大量的工作岗位，在邪恶阶段的核爆阶段毫无帮助，在播种阶段后期才有可能填满牧师的工作位，帮助不能说没有但是比较有限。且“信仰”系列的技能普遍价格较高，建议不要点太多，
解锁【信仰】即可，后期若是质粒剩余较多可以考虑【敬拜】。
④“超越”系列（【突变】）
【超越】是所有技能中加成最大的技能之一，也是流程最长，消耗最大的技能之一。在邪恶宇宙阶段的任务就是要在出邪恶之前点出“超越”系列前置中的【突变】。
【突变】有两条前置线：
（1）【遗传记忆】-【阿尼姆斯】-【神圣的回忆】-【黄金分割比】-【基因库】
（2）【形态发生】-【合成】-【核分裂】-【胞质分裂】
两条技能升级路线由于金星挑战的原因对实际游戏提升并不明显，仅做前置技能要求，当【基因库】和【胞质分裂】研究完成后可以解锁【突变】。
如果已经完成了上述的四条技能加点仍有质粒余裕，则可以考虑以下的技能：
⑤“存储”系列（【空间推理】-【空间优势】-【空间霸权】-【维度扭曲】）
同样的因为金星挑战的原因，存储系列1、2、3都没有实际提升且价格比较昂贵，而存储4【维度扭曲】则是可以在金星挑战下利用噬菌体增加存储上限的强力技能。
⑥“总督”系列（【空间推理】-【地理学家】-【建筑师】-【总督】）
总督拥有比较强力的正面buff以及同样明显的负面debuff，对于黑洞重置及以后的重置有较大帮助，对于邪恶阶段金爆金播帮助有限，如果有余力可以点一点。


7.为啥邪恶宇宙结束还要回标准宇宙？为啥要做衰变？为啥用炎魔做衰变？为啥要点出【突变】？
①标准宇宙金黑洞之后能获得特权“白洞”，衰变挑战完成后能获得特权“消散”，详询wiki，都是非常实用的特权。
②因为炎魔很强，几乎是最强的路人种族（路人的意思就是非自建种族），邪恶宇宙没有地狱星球，所以回标准宇宙之后才能选炎魔。
③因为炎魔在拥有两个T0级种族特质的前提下，也拥有很强的负面特质“可怖”，会让他无法进行交易，利用突变删除该特质能大幅度降低游戏难度。

基本目标：

2. 继续核爆挖矿，获取5000反质粒

简单说明：
这一步非常简单，挖矿，赚取足够多的反质粒跑路就完事了，用无星核爆的方式刷取5000反质粒。
另外值得一提的，无星核爆的脚本和我们平时通用的脚本设置完全不同，群里有反物质侏儒无星核爆的脚本参考，可以试试。

常见问题：
1.“出血效应”点完了，后续“同步原则”和“星体认知”要点下去吗？
可以，但是没必要，只有“出血效应”是必须在反物质宇宙用反质粒点的，剩下的可以等到高引力宇宙用质粒点。

2. 为什么是5000反质粒？
在后续步骤中，“星体认知”点完后，在其他宇宙中反质粒的软上限大约为噬菌体的四倍。按照萌新推荐路线流程，当高引力主舞台的诸多挑战任务完成后，
当时的噬菌体数量刚好5000反质粒左右可以达到软上限，之后带着更多的威望资源与特权再回到反物质宇宙，刷反质粒的效率将高得多，还可以顺便完成其他的专属任务。

3.反质粒多一点我看着舒服，可不可以多刷一点？
不拦着你，但是确实不建议。
反质粒刷取速度根据你的游戏进程会有非常非常明显的差异，比如刚出邪恶宇宙阶段反物质核爆一次需要800-1000天左右，当你高引结束回来可能只要百余天，
当你全永恒之柱镶嵌、智械黎明通关解锁机械族，回来甚至一次只要7-12天，这之间有近乎几十上百倍的效率差距，所以尽量每个阶段刷够需要的反质粒数量即可。

4. 无星核爆完需要把反物质宇宙的专属都金星完成吗？
同样，可以但是没必要。反物质宇宙的特性就代表着你不可能淹死，完成是一定能完成的，但是反物质宇宙的慢节奏也是不得不品尝的。
完不完成区别就是这些精通，作者本人是5000反质粒后直接无星黑洞跑路。

玩到这个阶段算是走上了萌新推荐路线的后期，面对最多的挑战，同时也应该拥有了比较深厚的游戏理解。

常见问题：

1. 主要流程的第二条看起来可能有点谜语人，什么叫全宇宙的全爆全播和高引力的全爆全播？
前者指的是，在总的成就下，所有物种都经历过一次毁灭和所有种群都经历过一次伟大,
这一步是为了“重量级拳皇”特权，以及后续的银飞捏人——只有和毁灭过的物种的特质以及伟大过的种群才能在捏人时被选取；
而后者指的是，在高引力宇宙成就中，所有物种和种群都要毁灭/伟大过一次，换句话说，你邪恶宇宙金炸过的种族在第二步还要再金炸一次,
这一步的目标是积攒本宇宙精通，为后续的挑战做铺垫。当然第二步不用完全严格执行，多少十几点精通不会有太过于明显的差距。

2. 银飞用什么种族比较合适？
就路人而言炎魔最强；精怪的树妖种族获取同族特质后拥有难以捉摸、音乐家和树木亲和三大T1特质；草族的水豚可以有冷静和射手。
总体而言银飞难度不高，选人不是特别关键。

3. 地震星球有什么说法？
为后续大灾变挑战提供场地，大灾变挑战必须在地震星球上才能进行

4. 关于金飞EM
目前金飞EM是有特权奖励的挑战中，耗时最高的挑战之一，算是前面那么长堆精通堆威望资源之后的毕业之战。预计耗时在八万天左右，祝各位好运。

5. 大灾变捏人有什么说法？
大灾变会失去地面所有建筑，除了资源生产建筑外，值得注意的是特质建筑同样会消失，诸如奴隶贩子、冷静等强力特质将失去作用，注意捏人的时候不要携带。
此外群里有专门的大灾变捏人攻略，特别提醒大灾变和大灾变飞升是完全不同的两种挑战，灾飞是壮举中数一数二困难的项目，在选择捏人攻略的时候记得观察文件名称。

6. 大灾变*2啥意思？
大灾变相关成就有两个，一个是灾中灾，即大灾变模式下触发大灾变重置，另一个是灾播。建议先灾中灾然后灾播。

7. 大灾变怎么禁脚本啊！
大灾变是不禁脚本的，但是假如你用正常流程，即建造世界对撞机之后，刻度盘拨到11进入大灾变模式，你会发现是没有脚本的。
但是此时你的基因挑战里已经解锁了大灾变模式，所以直接软重置然后重新进入大灾变模式，就可以用脚本了。
不过还是建议大灾变模式至少禁用自动建筑和自动研究。
